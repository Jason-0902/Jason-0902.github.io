Relational Algebra CLI Simulator

This project simulates relational algebra operations via a command-line interface written in Python. It follows SQLite-like syntax and is suitable for educational or prototyping purposes.

Supported Operations

  • CREATE TABLE
  • INSERT INTO
  • SELECT
  • PROJECT
  • RENAME
  • UNION
  • DIFFERENCE
  • INTERSECT
  • DIVIDE
  • JOIN (Natural Join)
  • SHOW, SCHEMA, USE

Example Usage

CREATE TABLE students (id, name, age)
INSERT INTO students VALUES (1, 'Alice', 20)
INSERT INTO students VALUES (2, 'Bob', 21)
SELECT * FROM students WHERE age > 20
PROJECT name, age FROM students
